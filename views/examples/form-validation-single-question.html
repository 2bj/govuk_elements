Error example

{{> form_error_single_question }}

When an error occurs:

  • add a 5px red left border to the field with the error
  • the label should be red and bold
  • the error message be red and bold and appear above the field
  • the error message should sit inside the <label>
  • keyboard focus should be moved to the input
  • consider using aria-invalid attribute to programmatically indicate that a field has an error
  • the value of the aria-invalid attribute will vary depending on the type of error
